    About

    DevSupply provides enterprise-grade tools written by developers, for developers. We let you focus on building value instead of plumbing.

    Solutions

    For Software and SaaS Providers

    Hub delivers professional grade integrations that allow you to meet your customers where they are, and gives them the control they need to achieve the highest levels of certification.

    All DevSupply products and solutions are available under OEM / white label licensing terms, allowing you to quickly offer advanced integrations features to your product with seamless branding.

    For IoT and Device Manufacturers

    Hub is the perfect platform to deliver a modern, RESTful control pane for your customers and partners. The hub agent framework handles device registration, secure communications, geographic scaling, and software updates.

    Your customers gain all the advanced Hub features such as API sharing, request logging, web based console, and OpenAPI support. All Hub features are available under OEM licensing terms on a per-device basis.

    For IT Managers

    Hub is the single pane of glass that gives you the ultimate control of every API in the company. Decide what teams, or what people, or what partners have access. Decide what operations are permitted, in which environments, in what time of day, and in what quantity. Hub natively supports PAMs and Secret Managers, making sure secrets are never shared to API consumers.

    Hub is also fully transparent, with a complete log of every API call in minute detail.

    For Low Code / No Code Platforms

    Hub enables unprecedented power to extend low-code and orchestration platforms by enabling a wide array of enterprise-ready modules, as well an allowing access to resources behind firewalls and the secure execution of code on hosts.

    Hub is the ultimate “last mile” technology that can safely reach the most challenging targets. The Hub SDK enables your customers to package all their custom scripts and apps into REST services that are fully described using OpenAPI.

    For Publishers

    Hub Marketplace allows for full monetization of your logic. Using the Hub SDK, you can develop, deploy, and monetize access or business logic, enabling rapid and secure delivery into some of the world’s leading IT shops.

    For DevOps

    Hub is the ultimate connector. It allows pipelines and automation to access every connected system in the company, as well as every team. The AHI feature of hub creates a simple but powerful interface to the business to automate approvals, tasks, escalations, and notifications. The PAM and Secrets integrations remove all the burdens for managing secrets, with automated workflows tied to your IT and compliance department to request secrets as necessary.

    For IT Compliance

    Hub is the ultimate command and control center for API access. Hub centralizes access controls, logging, reporting, and authorization requests.

    The Hub SDK is also the optimal way to package, version, and distribute internal scripts (Powershell, Python, Bash, etc), exposing them as callable REST services to orchestration and automation flows.

    Partner Access Management

    Hub us the perfect way to share least-privileged access to applications and resources with internal or external customers. Select partners can be given highly granular access, down to a single API call on a single module, enabling dozens of use cases including:

    • ETL and data interchange
    • IT Audit and Compliance
    • Product Support / Vendor Access Management
    • Outsourced Software Development

    For Digital Transformation

    The Hub suite tackles many of the most difficult challenges in the path to digital transformation, including:

    • Standardized access to all apps (SaaS or Legacy) and devices (servers, IoT, endpoints), even behind firewalls
    • Streamlined authentication, authorization, and approvals to make sure the right level of access is always available
    • PAM and Secrets Manager integration to make sure API consumers never see passwords, using only time and scope-limited API tokens.
    • Seamless integrations to work teams, allowing automation to open tasks and approvals in the tools where your Teams are already invested

    Products

    Hub (one product)

    Platform

    Command and Control for APIs

    Hub features a SaaS or customer hosted control plane that provides a single pane of glass for API governance. Hub Platform allows for highly granular API sharing, accounting, and cost management, with integrated approval workflows and detailed reporting.

    Modules

    Universal Packaging for Code and Connectors

    Modules are code bundles that were developed using the Hub SDK. Modules are written in JavaScript, TypeScript, Python, Java, or Powershell and then published to the Hub Marketplace. Modules can execute code, run scripts, or bridge to remote APIs such as REST, SOAP, SQL, SNMP, LDAP, etc. Every function in a module is presented as an OpenAPI operation once deployed.

    Marketplace / SDK

    The App Store for Modules

    The Hub Marketplace is the one-stop shop for connecting to hundreds of systems, products, and utilities. Shop from modules built by DevSupply, third party vendors, or your in house development team. The marketplace has advanced controls to enable the right level of access and consumption for different teams, with detailed usage reporting and chargeback capabilities.

    Agents

    Deploy Modules Anywhere

    Hub Agents are where Hub Module code is executed. Hub supported agents on traditional operating systems (Windows, Linux, MacOS) or inside virtual infrastructure (appliances, containers, Docker). Agents maintain FIPS-140 encryption for all data communications, and include advanced sandboxing features to prevent unauthorized system access. The lightweight agent library is slim enough to run in a wide variety of embedded platforms including Raspberry Pi and Android.

    AHI

    Automated Human Intelligence

    Hub’s AHI enables seamless collaboration between your people and automated workflows. AHI provides developers with REST APIs and callbacks for Tasks, Approvals, Files, and Notifications, reaching every team where they currently work, for example opening tickets for Development in Jira, IT in ServiceNow, and Marketing in Asana.

    Hydra

    Foundational Services

    Hydra is a collection of behind-the-scenes components that enables the Hub Platform, including a custom API gateway (Dana), distributed processing, session management, login screens, and identity brokerage. Hydra also includes multi-cloud, multi-region deployment and scaling based on Terraform and Helm.
