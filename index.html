    Tools For Developers, By Developers

    Hub Module Hub Agent Hub Platform AHI Hydra

    Hub Module

    Connection hub

    Unlocks access to a catalog of readily available systems and applications through a single API connection:

    • 2-way traffic to and from systems
    • No rev management
    • No API management

    Hub Agent

    The runtime environment that executes a Hub Module. Can be a Windows Service, a Linux Agent, Docker Container, or VM. Can run behind firewalls or endpoints.

    Hub Platform

    The presentation and authorization layer that gates all Hub Modules and Agents. Enables secure API sharing and accounting, as well as code distribution and versioning.

    AHI

    Automated Human Interaction

    Orchestration platform between all connected systems and teams (people as APIs) for approvals, notifications, and task management - meeting teams in the systems they prefer.

    Hydra

    Foundational Services

    Hydra is a suite of foundational services for modern, distributed serverless applications that includes an API gateway (Dana), identity services, login screens, invitation and registration functions, session management, data streaming, and Helm-based deployment utilities.
