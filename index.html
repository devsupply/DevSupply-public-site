    Platform

    Command & Control for APIs

    Hub features a SaaS or customer hosted control plane that provides a single pane of glass for API governance. Hub Platform allows for highly granular API sharing, accounting, and cost management, with integrated approval workflows and detailed reporting.

    Modules

    Universal Packaging for Code & Connectors

    Modules are code bundles that were developed using the Hub SDK. Modules are written in JavaScript, TypeScript, Python, Java, or Powershell and then published to the Hub Marketplace. Modules can execute code, run scripts, or bridge to remote APIs such as REST, SOAP, SQL, SNMP, LDAP, etc. Every function in a module is presented as an OpenAPI operation once deployed. Ultimately, Modules unlocks access to a catalog of readily available systems and applications through a single API connection:

    • 2-way traffic to and from systems
    • No rev management
    • No API management

    Marketplace

    The App Store for Modules

    The Hub Marketplace is the one-stop shop for connecting to hundreds of systems, products, and utilities. Shop from modules built by DevSupply, third party vendors, or your in house development team. The marketplace has advanced controls to enable the right level of access and consumption for different teams, with detailed usage reporting and chargeback capabilities.

    Agents

    Deploy Modules Anywhere

    Hub Agents are where Hub Module code is executed. Hub supported agents on traditional operating systems (Windows, Linux, MacOS) or inside virtual infrastructure (appliances, containers, Docker). Agents maintain FIPS-140 encryption for all data communications, and include advanced sandboxing features to prevent unauthorized system access. The lightweight agent library is slim enough to run in a wide variety of embedded platforms including Raspberry Pi and Android.

    AHI

    Automated Human Intelligence

    Hub’s AHI enables seamless collaboration between your people and automated workflows. AHI provides developers with REST APIs and callbacks for Tasks, Approvals, Files, and Notifications, reaching every team where they currently work, for example opening tickets for Development in Jira, IT in ServiceNow, and Marketing in Asana.

    Hydra

    Foundational Services

    Hydra is a collection of behind-the-scenes components that enables the Hub Platform, including a custom API gateway (Dana), distributed processing, session management, login screens, and identity brokerage. Hydra also includes multi-cloud, multi-region deployment and scaling based on Terraform and Helm.
