    DevSupply Platform

    The presentation and authorization layer that gates all Hub Modules and Agents. Enables secure API sharing and accounting, as well as code distribution and versioning. The platform delivers 3 core product offerings:

    1. Hub
    2. AHI
    3. Hydra

    Tools For Developers, By Developers

    Platform Modules Marketplace Agents AHI Hydra

    Platform

    Hub features a SaaS or customer hosted control plane that provides a single pane of glass for API governance. Hub Platform allows for highly granular API sharing, accounting, and cost management, with integrated approval workflows and detailed reporting.

    Modules

    Connection hub

    Modules are code bundles that were developed using the Hub SDK. Modules are written in JavaScript, TypeScript, Python, Java, or Powershell and then published to the Hub Marketplace. Modules can execute code, run scripts, or bridge to remote APIs such as REST, SOAP, SQL, SNMP, LDAP, etc. Every function in a module is presented as an OpenAPI operation once deployed. Ultimately, Modules unlocks access to a catalog of readily available systems and applications through a single API connection:

    • 2-way traffic to and from systems
    • No rev management
    • No API management

    Marketplace

    Agents

    The runtime environment that executes a Hub Module. Can be a Windows Service, a Linux Agent, Docker Container, or VM. Can run behind firewalls or endpoints.

    AHI

    Automated Human Interaction

    Orchestration platform between all connected systems and teams (people as APIs) for approvals, notifications, and task management - meeting teams in the systems they prefer.

    Hydra

    Foundational Services

    Hydra is a suite of foundational services for modern, distributed serverless applications that includes an API gateway (Dana), identity services, login screens, invitation and registration functions, session management, data streaming, and Helm-based deployment utilities.
